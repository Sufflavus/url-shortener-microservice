URL Shortener Microservice

Example creation usage:

https://tata-fcc-url-shortener.herokuapp.com/new/https://www.google.com
https://tata-fcc-url-shortener.herokuapp.com/new/http://foo.com:80

Example creation output:

{ "original_url":"http://foo.com:80", "short_url":"https://tata-fcc-url-shortener.herokuapp.com/8170" }

Usage:

The url

https://tata-fcc-url-shortener.herokuapp.com/1

Will redirect to

http://stackoverflow.com/
